This is a full-stack web application built with **Node.js, Express, MongoDB Atlas, and EJS**.  

It allows users to:

- Register and log in with authentication  
- Create, read, update, and delete listings  
- Upload images with Cloudinary  
- Write and manage reviews  
- Persist sessions securely with MongoDB  

This project demonstrates **real-world full-stack development** skills, including handling sessions, authentication, file uploads, and deployment-ready configurations.

---

## ⚡ Features
- User authentication with **Passport.js**  
- Session management via **express-session + connect-mongo**  
- **CRUD operations** for listings  
- **Image uploads** with Multer + Cloudinary  
- Geocoding addresses to coordinates  
- Flash messages for user notifications  
- Custom error handling using `ExpressError`  
- MongoDB Atlas cloud database integration  

---

## 🛠️ Technologies Used
- **Backend:** Node.js, Express  
- **Database:** MongoDB Atlas, Mongoose  
- **Authentication:** Passport.js, express-session  
- **File Upload:** Multer, Cloudinary  
- **Frontend:** EJS, ejs-mate  
- **Other Packages:** connect-mongo, connect-flash, method-override, dotenv
